# SecuriteAndroid
Analyse et sécurisation d’une application Android — de la récupération jusqu’au reporting et à la politique mobile.

Objectifs pédagogiques :

Savoir récupérer un APK vulnérable, faire de l’analyse statique (manuelle + outils automatisés), décompiler/recompiler/Signer un APK, détecter fuites (strings.xml, Firebase…), produire un rapport de vulnérabilités et définir des mesures organisationnelles (politique mobile + choix MDM).
